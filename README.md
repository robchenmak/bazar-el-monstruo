# Bazar el Monstruo 🛍️

Sitio web estático para el **Bazar el Monstruo**, un proyecto personal para mostrar productos, promociones y permitir pedidos vía WhatsApp.

---

## 📂 Estructura del proyecto
- `index.html` → Página principal con nombre del bazar y datos de contacto.
- `ubicacion.html` → Mapa con coordenadas de la ubicación del bazar.
- `galeria.html` → Galería de productos.
- `promociones.html` → Página de promociones.
- `carrito.html` → Carrito de compras con botón de pedido vía WhatsApp.
- `styles.css` → Estilos generales.
- `script.js` → Lógica básica para galería y carrito.
